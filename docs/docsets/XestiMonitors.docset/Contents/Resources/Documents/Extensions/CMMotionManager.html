CMMotionManager

Undocumented

  • shared

    Returns the singleton motion manager instance.

    According to Apple:

    An app should create only a single instance of the CMMotionManager class. Multiple instances of this class can affect the rate at which data is received from the accelerometer and gyroscope.

    By default, all motion monitor classes use this shared motion manager instance.

    Declaration

    Swift

    public static let shared = CMMotionManager()
